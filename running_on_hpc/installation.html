InstallationÔÉÅ

We will describe how to transfer files through WinSCP and submit jobs through PuTTY, but it is possible to access the clusters in many ways, such as through Visual Studio Code or ThinLinc.

Software Requirements

  • Download and install WinSCP for transferring files to the HPC.

  • Download and install PuTTY for submitting jobs.

  • Download and install Cisco VPN to access DTU network from home.
